tw
--------

[Description]

Install:

    npm install tw
	
Add dependency to your project with

    "tw": "git@github.com/Michieljoris/tw"
	
or

	"tw": "*"

Require in a module:

    var tw = require('tw');

Use:

See [documentation](https://rawgithub.com/Michieljoris/tw/master/docs/tw.html).
